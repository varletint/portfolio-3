Instructions to Generate OG Image:

  1. Open this HTML file in a browser (Chrome recommended)
  2. Right-click on the colored rectangle above
  3. Select "Inspect" or "Inspect Element"
  4. In DevTools, right-click on the <div id="og-image"> element
  5. Select "Capture node screenshot" (Chrome) or use a screenshot tool
  6. Save the image as og-image.png
  7. Move the file to /public/og-image.png

Alternative Method - Using Browser Screenshot:

  1. Use a browser extension or online tool to capture the exact dimensions
  2. Recommended tools:
    • Chrome DevTools (Capture node screenshot)
    • Firefox Screenshot tool
    • Online: Figma (free)
    • Online: Canva (free)

Verification:

  • Image dimensions: 1200x630 pixels
  • Background color: #faf6f1
  • File size: Should be under 500KB
  • Format: PNG
